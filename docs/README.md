# Trading Bot 가이드 문서

이 프로젝트는 암호화폐 시장 데이터를 실시간으로 수집하고, 수집된 데이터를 바탕으로 트레이딩 전략을 시뮬레이션(백테스트)할 수 있는 고성능 트레이딩 봇 프레임워크입니다.

## 🚀 프로젝트 요약
`trading-bot`은 Binance Futures 및 Alternative.me(Fear & Greed Index) 등 다양한 소스로부터 시장 데이터를 수집하여 고성능 시계열 데이터베이스인 QuestDB에 저장합니다. 또한, 저장된 과거 데이터를 기반으로 다양한 트레이딩 전략을 정교하게 검증할 수 있는 이벤트 기반 백테스팅 엔진을 제공합니다.

## 🏗️ 핵심 아키텍처 요약
본 프로젝트는 크게 **Collector(수집기)**와 **Backtester(백테스터)** 두 부분으로 나뉩니다.

- **Collector**: `tokio` 비동기 런타임을 활용하여 다중 WebSocket 스트림을 동시에 유지하며, 수집된 데이터를 `mpsc` 채널을 통해 중앙 집중형 `QuestDB Writer`로 전달하여 효율적으로 저장합니다.
- **Backtester**: QuestDB로부터 과거 데이터를 대량으로 인출(Fetch)하여 시간 순으로 정렬(Merge)한 뒤, 이를 전략(Strategy)에 주입하는 이벤트 기반 시뮬레이션을 수행합니다. 실제 거래와 유사한 환경을 위해 호가창(Order Book) 기반의 체결 엔진과 수수료 모델을 포함합니다.
